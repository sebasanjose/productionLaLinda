# La Linda Empanada Tracker

[![Python Tests](https://github.com/sebasanjose/productionLaLinda/actions/workflows/python-tests.yml/badge.svg)](https://github.com/sebasanjose/productionLaLinda/actions/workflows/python-tests.yml)

A Flask application for tracking empanada production, inventory, and market sales.

## Getting Started

### Prerequisites

- Python 3.9+
- Docker and Docker Compose
- PostgreSQL database

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/sebasanjose/productionLaLinda.git
   cd productionLaLinda
   ```

### Continuous Integration

This project uses GitHub Actions for continuous integration:

- Tests are automatically run on every push and pull request
- The workflow includes running pytest and generating coverage reports
- You can view the test results in the Actions tab of the GitHub repository
